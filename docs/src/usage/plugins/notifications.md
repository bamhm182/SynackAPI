# Notifications

## notifications.get()

> Return a list of notifications
>
>> Examples
>> ```python3
>> >>> h.notifications.get()
>> [{"action": "outage_starts", "subject": "SLAPHAPPYMONKEY",...}...]
>> ```

## notifications.get_unread_count()

> Get the number of unread notifications.
>
>> Examples
>> ```python3
>> >>> h.notifications.get_unread_count()
>> 7
>> ```
